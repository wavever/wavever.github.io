请回答 2019

发表于 2020-01-04 | 分类于 碎碎念

拖延症患者的本命年总结。

本来打算写在 2019 的最后一天，但 31 号那天下班回到家已经是晚上 11 点多了，庆幸没有解锁“公司跨年”的成就，却也很真实的反映了近半年以来的现况，那就是一个字：忙。

回顾

大部分时间和2018没有什么大区别，没钱人的生活就是这么的重复且重复。双休日变成了“双休日”，995变成了10115.5，属于自己的时间少了，休息时间基本都是宅在家里逛逛B站，打打游戏，发发呆。

从离家两千公里的魔都，来到了离家一千公里的成都，火车上睡一觉就到家的设想变成现实。

5月初从工作了两年的传音来到了绿厂，年轻人的第一次跳槽，整个过程目前想来还是有点心累，疯狂恶补各种面试所需资料，每天晚上熬早上早起复习，那一段时间真是难熬，差点就选择了放弃。还好最后拿到了小米和绿厂 Offer，考虑了很久，选择了离家更近的成都，作为一个初级米粉，很遗憾没有成为小米的一员。

看了几部电影，《中国机长》真的顶，《攀登者》有点失望。

断断续续的总算是把《Android 开发艺术探索》里所有的章节都过了一遍，《深入理解 Java 虚拟机》只阅读了垃圾回收的相关内容，《Android 源码设计模式解析与实战》阅读进度 1/4。

时隔两年后首次回学校，只可惜这次已经没有宿舍可以回去了，不得不说，在重庆吃火锅还是别有一番风味的。

展望

虽说现在自己现在也在怀疑，每天都这么忙，更多的时候也都是处理一些琐事 ，到底能给自己带来什么？但时间久了还是发现，学习是自己的事情，在做好本职工作的同时去提升自己的能力，才是目前自己更应该关心的。

2020 年，在自己工作的第三年，希望自己能够改善拖延症的习惯，可以在技术的深度和广度上有所进步，在这里先给自己定一些小目标，也算是一些警戒。

  • 阅读 3 本及以上的技术书籍；
  • 业余时间，完成一款应用的开发，已经知道要做啥了，接下来就看自己的行动力了，希望明年回顾的时候不会打脸；
  • 了解一点前端以及后端的知识，能入门最好；
  • 保持写博客的习惯；
  • 改善自己宅的习惯，多出去走走，世界这么大，假期可以去看看。

最后，以最喜欢的一句话来结束本文：

永远相信美好的事情即将发生
