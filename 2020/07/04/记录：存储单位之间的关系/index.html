记录：存储单位之间的关系

发表于 2020-07-04 | 分类于 混淆点记录

最近经常容易搞不清这些存储单位之间的关系，俗话说，好记性不如烂笔头，记录一下，防止下一次又忘了，尴尬😅

bit

位、比特，计算机中最基本的存储单元，二进制中的 0 或 1 即为一个比特。

byte

字节，1 byte = 8 bit。

KB

千字节，1 kb = 2^10 byte = 1024 byte。

MB

兆字节，1 MB = 2^20 byte = 1024 KB
