Hexo 博客记录

发表于 2018-08-26 | 分类于 Hexo

最近换了个 mac😄，因为之前没有用过 macOS 系统，所以包括搭建开发环境和安装一些软件都得去搜索如何安装，本篇博客主要记录一下如何在换机器或是重装系统时转移博客内容，防止下次需要用时还要到处搜索。

安装 Nodejs

博客是使用 Hexo 搭建，因此首先需要去按转 NodeJs ，在 macOS 上，可以选择使用官网下载压缩包或是使用 homebrew 安装：

homebrew install node

安装 Hexo

npm install hexo-cli -g
进入到想生成博客的文件夹或是填入路径：

hexo init

或是hexo init path

生成静态文件：hexo generate

安装主题

安装 next 主题

$ git clone --branch v5.1.2 https://github.com/iissnan/hexo-theme-next themes/next
主题可以通过更改配置文件来修改配置，如果之前已经使用过该主题，直接可以将之前的配置文件拷贝过来进行覆盖，不过如果之前主题版本较老，还是需要和新版本主题配置文件对比一下的。

转移博客内容

复制 _post 文件夹至新地址进行覆盖，如果有关于或是分类等页面，可以生成对应 hexo new page "about" 页面后复制之前的 index.md 覆盖。

生成、部署博客

使用命令：
hexo g 以及 hexo d

或是直接使用：
hexo d -g
此时，如果出现错误：error deployer not found:git ，需要去安装 git 部署工具：
npm install hexo-deployer-git --save


大功告成～可以在新机器上愉快的玩耍了😄
